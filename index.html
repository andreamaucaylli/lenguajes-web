Algunos lenguajes de la web

HTML+CSS+Javascript

  • HTML 5

    • HTML5 (HyperText Markup Language, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML.

    Usa la extensión .html

  • CSS

    • Hoja de estilo en cascada o CSSb> (siglas en inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentación de u documento estructurado escrito en HTML.

    Usa la extensión .css

  • JavaScript

    • JavaScript (abreiado comunmente "JS") es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejora

    Usa la extensión .jss
